# 测试报告：客户端向服务器发送1000次请求

## 测试目标

客户端通过 TUIC 代理服务器，向 `https://kh.google.com/rt/earth/PlanetoidMetadata` 发送 **1000次请求**

## 测试配置

- **目标URL**: `https://kh.google.com/rt/earth/PlanetoidMetadata`
- **总请求数**: 1000
- **并发数**: 50
- **代理端口**: 18443
- **超时时间**: 60秒

## 测试步骤

### 1. 启动代理服务器

```bash
./utlsProxy -listen 127.0.0.1:18443 -token test-token-12345 \
    -cert server.crt -key server.key -log info
```

### 2. 运行并发测试

```bash
./utlsclient-cli \
    -concurrent-test \
    -proxy 127.0.0.1:18443 \
    -token test-token-12345 \
    -url "https://kh.google.com/rt/earth/PlanetoidMetadata" \
    -concurrency 50 \
    -requests 1000 \
    -timeout 60s
```

### 3. 使用测试脚本（一键运行）

```bash
./test_1000_requests.sh
```
