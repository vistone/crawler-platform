## 版本说明

### 版本号格式

版本号格式: `MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的API变更
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修复

### 如何升级

#### 从 0.0.14 升级到 0.0.15

本次升级为文档更新，无需代码变更。

#### 从 0.0.13 升级到 0.0.14

本次升级包含重要性能优化和新功能:

1. **HTTP/2 支持**: 无需代码变更，自动启用
2. **IPv6 支持**: 无需代码变更，自动检测
3. **Accept-Language**: 自动添加到请求头
4. **配置调整**: 根据需要调整 `config.toml` 中的配置项

```bash
# 更新依赖
go mod download

# 重新编译
go build ./...

# 运行测试
go test ./...
```
